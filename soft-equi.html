I use free/libre software. Proprietary software shouldn't exist and the users should be more aware about it (FSF).

Software I use

  • Operating System:
    GNU/Linux (I didn't write Linux). I used several distros during the years and I can say my favorite distribution is Arch Linux.
  • Window Manager:
    i3-gaps. It's a fork of i3wm, a tiling window manager. Why should I use a Desktop Environment like GNOME, KDE, XFCE and so forth ? Tiling window managers are faster (you can do all without mouse), simpler and completely configurable.
  • Terminal:
    urxvt. It's good but I'm not completely satisfied so I'm looking for something better.
  • File manager:
    ranger. It's a console file manager well-featured and customizable but a little bit slow being implemented in python.
  • Text editing:
    emacs. It's the best software I have never used so far. If I need to edit a file quickly and I don't need of all the emacs power I use Vim (why not?).
